Smart Features That Work Around the Clock

Our intelligent system monitors and controls your greenhouse environment automatically

Temperature Control

Precise temperature monitoring with LM35 sensor. Automatic cooling fan activation when temperature exceeds 30¬∞C to maintain optimal growing conditions.

Sensor: LM35 (¬±0.5¬∞C accuracy)

Smart Irrigation

Soil moisture detection with automatic watering system. Pump activates when soil moisture drops below optimal levels (400 units).

Trigger: Moisture < 400 units

Light Management

Photoresistor-based light sensing with automated curtain control. Opens curtains when light levels drop below 300 units for optimal photosynthesis.

Threshold: Light < 300 units

Real-time Display

16x2 LCD screen shows live sensor readings. Clear display of temperature, soil moisture, and light levels updated every 3 seconds.

Update Rate: Every 3 seconds

Automated Control

Three independent actuators control irrigation pump, cooling fan, and curtain motor based on real-time sensor feedback.

Outputs: 3 Digital pins (5V)

Serial Monitoring

Debug and monitor system performance through serial communication. Track all sensor readings and system decisions in real-time.

Baud Rate: 9600 bps
